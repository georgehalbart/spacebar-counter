# Spacebar Counter

A simple and lightweight browser extension that allows users to test how fast they can press the spacebar in real time.

## 🚀 Features
- Real-time spacebar press counter
- Instant and accurate feedback
- Clean and minimal interface
- Lightweight and fast performance
- Works offline

## 🎯 Use Cases
- Test keyboard responsiveness
- Improve typing speed
- Practice reaction time
- Fun speed challenges

## 🛠️ How to Use
1. Install the extension
2. Open the extension popup
3. Start pressing the spacebar
4. View your total count instantly

## 🔒 Privacy
This extension does not collect, store, or transmit any personal or sensitive user data.  
All functionality runs locally in the browser.
